📌 Project Overview

In today's competitive retail environment, businesses need more than reports—they need data-driven insights that support faster and smarter decisions.

This project analyzes Coffee Shop Sales data using MySQL for data preparation and Power BI for visualization. The dashboard helps business leaders monitor performance, identify trends, optimize operations, and improve profitability through interactive analytics.

🚀 Business Problem

Coffee shop management wanted to answer critical business questions:

How much revenue is the business generating?
Are sales increasing or decreasing?
Which products generate maximum revenue?
Which store performs the best?
What are customers buying most?
Which days and hours generate maximum sales?
How can low-performing periods be improved?

This dashboard provides answers through interactive visualizations and KPIs.
